Exemplo de Uso do Widget Simulador Fio B

Este é um exemplo de como incorporar o widget Simulador Fio B em seu site.

Como usar:

  1. Adicione um elemento <div id="simulador"></div> onde deseja que o widget apareça
  2. Inclua o script loader antes do fechamento da tag </body>: 
    <script src="https://seu-dominio.com/loader.js"></script>

Nota: Substitua https://seu-dominio.com pela URL real onde os arquivos estão hospedados.
